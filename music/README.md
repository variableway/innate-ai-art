## 快速开始（10分钟出第一首歌）

如果你想跳过教程直接上手，按这个最短路径操作：

1. 打开 [Suno](https://suno.com)（免费注册，每天50积分）
2. 在 Style 框输入：`Indie Pop, dreamy, warm synth pads, 120 BPM, in C major`
3. 在 Lyrics 框输入：`[Verse]` 换行后写2-4行中文歌词
4. 点击 Create，等待1分钟
5. 听结果，不满意就改一个词重新生成

## 关键心态

- **生成10版选最好的**：不要期待一次满意，AI 的"意外"有时更好
- **解构而非模仿**：提取速度+调性+情绪+乐器组合，重新组合
- **迭代纪律比天赋重要**：专业创作者平均 7-12 轮迭代，不要在第 1-2 轮就放弃
