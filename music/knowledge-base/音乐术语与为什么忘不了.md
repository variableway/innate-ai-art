## 第一部分：术语速查

### 1.1 Riff / Hook / Motif / Lick 的区别

这些是音乐创作中最容易混淆的四个词，一张图搞清：

```
记忆性 强 ←────────────────────────→ 弱
          Hook          Riff          Lick
          (任何形式)     (乐器)        (即兴)
          │
          ├── 可以是歌词   ── 反复出现 ──  出现一次
          ├── 可以是旋律    ── 歌曲签名 ──  乐手炫技
          └── 可以是节奏

          Motif
          (2-5个音的种子)
          │
          └── 整首歌从这几颗种子"长"出来
```

| 术语 | 中文 | 含义 | 特点 | 例子 |
|------|------|------|------|------|
| **Riff** | 即兴重复段 | 反复出现的乐器短旋律 | 乐器的签名，2-4 小节 | "Just Like Heaven" 开头吉他 |
| **Hook** | 钩子/记忆点 | 让你忘不了的部分 | 不限形式，最容易被记住 | "We Will Rock You" 的节奏 |
| **Motif** | 动机 | 2-5 个音的旋律种子 | 最短，是发展的基础 | 贝多芬第五交响 "登登登邓" |
| **Lick** | 乐句 | 即兴演奏的装饰短句 | 通常只出现一次，炫技用 | 吉他 Solo 中间的花式段落 |
| **Vamp** | 反复段 | 简单和弦/节奏循环 | 伴奏背景 | Funk 歌曲的节奏吉他 |
| **Ostinato** | 固定音型 | 不断重复的旋律/节奏型 | 古典音乐术语 | Pachelbel's Canon 低音线 |

### 1.2 歌曲结构术语

| 术语 | 中文 | 含义 |
|------|------|------|
| **Intro** | 前奏 | 歌曲开头，通常纯器乐 |
| **Verse** | 主歌 | 讲故事、铺垫 |
| **Pre-Chorus** | 预副歌 | 主歌和副歌之间的爬升过渡 |
| **Chorus** | 副歌 | 情感爆发、最容易被记住的段落 |
| **Bridge** | 桥段 | 和前面不同的转折 |
| **Outro** | 尾奏 | 收尾 |
| **Breakdown** | 分解段 | 突然变安静（然后通常再爆发） |
| **Drop** | 坠落/爆发 | 能量突然释放的点 |
| **Fill** | 填充 | 段落过渡时的装饰音 |
| **Turnaround** | 回转 | 一段末尾引导回到开头的过渡 |

## 第二部分：为什么某些旋律让人忘不了？

### 2.1 科学解释：耳虫现象 (Earworm)

科学上叫 **INMI（Involuntary Musical Imagery，不自主音乐意象）**——旋律不由自主地在脑中反复播放。

```
大脑处理音乐的过程：

耳朵听到旋律
    ↓
听觉皮层处理音高/节奏
    ↓
海马体（记忆中心）存储旋律模式
    ↓
杏仁核（情绪中心）关联情感
    ↓
伏隔核（奖赏中心）释放多巴胺 → 感到愉悦
    ↓
大脑想要重复这个愉悦 → 耳虫循环开始
```

**哈佛医学院 David Silbersweig 的研究**指出：耳虫发生时，大脑的活动模式和你**真正在听音乐时几乎一样**。也就是说，你的大脑在"假装播放"这首歌——而且它乐在其中。

### 2.2 "忘不了"的 6 个核心特征

分析了几百首"耳虫级"歌曲后，它们共享 6 个特征：

#### 特征 1：极简——大脑能一次装下

```
能记住的旋律 = 7 ± 2 个音符组成的短句

"Just Like Heaven" Riff：约 8 个音符
"Seven Nation Army" Riff：7 个音符
"We Will Rock You"：只是一个节奏模式（0 个音符！）
"Tomorrow Never Knows" Beatles：5 个音符

规则：如果你不能在 3 秒内哼出来，它就太复杂了
```

> **为什么**：人的短期记忆容量约 7 个单元。一个 Riff 如果超过 7-9 个音，大脑就存不下来了。最洗脑的旋律通常只有 4-7 个音。

#### 特征 2：重复——大脑的"舒适区"

```
耳虫歌曲的重复模式：

"Just Like Heaven"：A-E-Bm-D 和弦进行从头到尾重复（约 16 次）
"Get Lucky" Daft Punk：同样的 4 小节循环约 30 次
"Billie Jean" Michael Jackson：贝斯线循环约 40 次

规则：重复 3 次以内 = 学习，重复 3-7 次 = 记住，重复 7 次以上 = 忘不了
```

> **为什么**：大脑是预测机器。当你能预测下一个音符时，大脑会释放多巴胺（"我猜对了！"的快感）。重复让预测变容易，快感增强。但完全可预测又会无聊——所以需要特征 3。

#### 特征 3：微小的意外——在熟悉中藏一个惊喜

```
"熟悉 + 微小意外" 公式：

"Just Like Heaven"：
  大调进行 I-V-？-IV → 期待 vi（小调）→ 实际给了 ii（Bm）→ 微小意外！
  为什么意外：大多数流行歌用 vi，The Cure 用了 ii

"Yesterday" Beatles：
  Fmaj7 和弦中的 E 音（本不该出现在 F 大调和弦里）→ 微小意外

"Creep" Radiohead：
  G-B-C-Cm → Cm 是"借来的"和弦，不在调内 → 意外！

规则：80% 可预测 + 20% 意外 = 最洗脑
```

> **为什么**：完全可预测 = 无聊。完全意外 = 混乱。大脑最喜欢的状态是"几乎猜对了但没完全猜对"。这个"差点就对了"的感觉让大脑反复回放，试图"搞清楚"。

#### 特征 4：节奏不对称——打破大脑的预测

```
对称旋律（无聊）：    | XX XX | XX XX |
不对称旋律（洗脑）：  | XXX X | XX XXX |

"Just Like Heaven" 副歌节奏：
"I'm in love with YOU" → | XX XX X | X |
比均匀的 | XX XX XX | 多了一点不对称

"Bohemian Rhapsody"：
节奏不断变化，你永远不知道下一个重音在哪

规则：在三拍子的歌里放两拍子，或在四拍子里用三连音
```

> **为什么**：不对称的节奏让大脑"算不准下一拍"，于是反复尝试预测，形成耳虫循环。

#### 特征 5：音程跳跃——旋律的"表情"

```
最洗脑的音程（两个音之间的距离）：

纯四度（Perfect 4th）：
  "Here Comes the Bride" → 开头的两个音
  "A Maiden's Prayer" → 开头
  听感：坚定、明亮

大三度（Major 3rd）：
  "Oh! Susanna" → 开头
  听感：快乐、温暖

小三度（Minor 3rd）：
  "Greensleeves" → 开头
  听感：忧伤、温柔

纯五度（Perfect 5th）：
  "Star Wars Theme" → 开头
  听感：英雄、史诗

八度跳跃（Octave）：
  "Somewhere Over the Rainbow" → "Some-WHERE"
  The Cure "Just Like Heaven" 副歌的跳高
  听感：突然的释放、超脱感
```

> **为什么**：特定的音程跳跃触发特定的情感。八度跳跃=超脱，小三度=忧伤，纯五度=英雄。最洗脑的旋律通常有一个标志性跳跃，用音程"表情"编码了情感。

#### 特征 6：与歌词/情绪绑定——意义让记忆更牢

```
旋律 + 情感 + 个人记忆 = 最强的耳虫

为什么你忘不了某首歌：
- 它在你初恋时播放过 → 杏仁核标记为"重要"
- 它的歌词说了你说不出的话 → 认同感
- 它的旋律和你当时的情绪共振 → 神经绑定

"Just Like Heaven" 的绑定：
旋律在 "I'm in love" 达到最高点 → 旋律高点和情感高点重合
→ 大脑把"恋爱的感觉"和"这个旋律"焊在一起了
```

> **为什么**：Cleveland Clinic 的研究指出，与情绪/记忆关联的歌曲形成耳虫的概率高 3-4 倍。大脑不会记住孤立的音符，但会记住"有意义的音符"。

### 2.3 特征总结——"忘不了"的公式

```
忘不了 = 简单 + 重复 + 微意外 + 不对称节奏 + 标志性音程跳跃 + 情感绑定

                     ┌─── 简单（7±2 个音）
                     ├─── 重复（3 次以上）
忘不了的音乐 ────────├─── 微意外（80/20 法则）
                     ├─── 节奏不对称
                     ├─── 标志性跳跃（音程表情）
                     └─── 情感绑定（旋律+歌词+记忆共振）
```

|------|------|------|--------|-----------|---------|---------|
| **The Cure "Just Like Heaven"** | 4 和弦 | 16 次 | ii 代替 vi | 副歌节奏偏移 | 八度跳高 | 恋爱感 |
| **Suede "Beautiful Ones"** | 4 和弦 | 12 次 | sus4 和弦 | 附点节奏 | 大六度 | 叛逆青春 |

### 2.5 把公式翻译成 AI 提示词

```
从"忘不了公式"提取的 AI 提示词模板：

[simple 4-chord progression], [repeating melodic riff],
[80% predictable with 20% surprise element],
[syncopated rhythm or slight rhythmic asymmetry],
[one signature interval leap in the chorus],
[melody peak aligned with lyrical emotional peak],
[catchy, memorable, earworm quality]

具体例子：

"Just Like Heaven 风格"：
A major, 150 bpm, I-V-ii-IV progression (surprising ii instead of vi),
simple 8-note guitar riff repeated throughout,
chorus melody jumps an octave at emotional peak,
syncopated vocal rhythm in chorus,
gradual layer building,
chorus-drenched shimmering guitar,
dreamy romantic yearning, earworm melody

"Suede 风格"：
A minor, 125 bpm, i-VII-IV-III progression with sus4 embellishments,
arpeggiated guitar riff as main hook,
dramatic falsetto vocal leap in chorus,
slightly asymmetrical phrasing,
theatrical emotional delivery,
crunchy overdriven guitar tone,
urban melancholy with glam energy, catchy hook
```

### 2.6 创作时的自查清单

写完一首歌（或 AI 生成一首歌）后，用这个清单检查它"能不能让人忘不了"：

```
□ 简单性：核心 Riff 能不能在 3 秒内哼出来？
□ 重复性：主要旋律至少出现了 3 次吗？
□ 微意外：有没有至少一个"跟预期不太一样"的音符/和弦？
□ 节奏：有没有一点不对称或切分，而不是死板的均匀？
□ 跳跃：副歌旋律有没有比主歌跳得更高或更低？
□ 情感绑定：旋律最高点和歌词情感最高点是同一个位置吗？
□ 整体：如果让朋友听一遍，他 10 分钟后还能哼出哪个部分？

如果 7 个都是 YES → 你有一首潜在的耳虫
如果 4-5 个 YES → 还行，可能需要加强重复或简化旋律
如果少于 4 个 → 考虑简化旋律、增加重复、或加一个意外元素
```
