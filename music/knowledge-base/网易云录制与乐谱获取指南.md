# 网易云音乐录制 & 乐谱获取

## 一、录制网易云音乐歌曲到本地

有几种常见方法：

### 方法 1：OBS Studio 录制系统音频（推荐）

OBS Studio 是免费开源的录制软件，Mac 和 Windows 都能录制系统音频（即电脑播放的所有声音）。

#### 1.1 安装 OBS Studio

**Mac**：
```bash
brew install --cask obs
```
或从官网 [obsproject.com](https://obsproject.com) 下载安装包。

**Windows**：
从官网 [obsproject.com](https://obsproject.com) 下载安装包，或：
```powershell
winget install OBSProject.OBSStudio
```

#### 1.2 Mac 配置系统音频捕获

Mac 默认无法直接录制系统音频，需要借助插件：

**安装插件（二选一）**：
```bash
# 方式 A：安装 BlackHole（免费虚拟音频驱动）
brew install blackhole-2ch

# 方式 B：安装 OBS 的 macos-capture 插件（推荐，更简单）
# 从 https://github.com/obs-ai/obs-macos-capture/releases 下载最新版
# 双击安装后重启 OBS
```

**如果使用 BlackHole，需配置多输出设备**：
1. 打开「应用程序 → 实用工具 → 音频 MIDI 设置」
2. 左下角点击 `+` → 「创建多输出设备」
3. 勾选你的扬声器/耳机 + BlackHole 2ch
4. 重命名这个多输出设备为「OBS 录音输出」
5. 在系统声音设置中将输出设备改为「OBS 录音输出」

**OBS 中添加音频源**：
1. 打开 OBS Studio
2. 在「来源」（Sources）面板点击 `+`
3. 选择「音频输出捕获」
4. 命名为「系统音频」→ 确定
5. 设备选择：
   - 如果装了 BlackHole：选 BlackHole 2ch
   - 如果装了 macos-capture 插件：选「屏幕音频捕获」或对应设备
6. 点击「确定」

#### 1.3 Windows 配置系统音频捕获

Windows 配置更简单，OBS 原生支持：

1. 打开 OBS Studio
2. 在「来源」（Sources）面板点击 `+`
3. 选择「音频输出捕获」（Audio Output Capture）
4. 命名为「系统音频」→ 确定
5. 设备选择你的扬声器或「默认」
6. 点击「确定」

#### 1.4 OBS 录制设置

**设置录制格式（重要）**：

1. 菜单栏 `设置` → `输出`
2. 输出模式选择「高级」
3. 切换到「录像」标签页：
   - **录像格式**：选择 `mkv`（录制中断也不会损坏文件，录完再转为 mp4）
   - **视频编码器**：`x264`（通用）或 `Apple VT H264`（Mac 硬件编码）
   - **音频编码器**：`AAC`
   - **比特率（音质）**：`320` Kbps（高品质）
4. 切换到「音频」标签页：
   - **采样率**：`48 kHz`
   - **声道**：`立体声`

**只录音频（不需要画面）**：

1. `设置` → `输出` → `录像` 标签
2. 视频编码器可以保持默认（不影响音频质量）
3. 或者在 OBS 场景中不添加任何视频源，只添加音频源
4. 录完后文件中会有音频轨，用工具提取即可

#### 1.5 开始录制

1. 在 OBS 主界面确认音频源已添加，音量表有波动
2. 打开网易云音乐，播放你想录制的歌曲
3. 点击 OBS 右下角的 **「开始录制」** 按钮
4. 歌曲播放完毕后，点击 **「停止录制」**
5. 录制文件默认保存在 `~/电影/OBS`（Mac）或 `视频/OBS`（Windows）

> **提示**：录制前先把网易云音量调到 100%，确保最佳音质。同时关闭其他可能发声的应用（微信、浏览器其他标签页等），避免杂音。

#### 1.6 从录制文件中提取纯音频

录制得到的是 `.mkv` 视频文件（即使只有音频），需要提取为纯音频：

```bash
# 安装 ffmpeg（如果还没有）
brew install ffmpeg  # Mac
# 或从 ffmpeg.org 下载 Windows 版

# 提取音频为 WAV（无损）
ffmpeg -i "录制文件.mkv" -vn -acodec pcm_s16le "歌曲名.wav"

# 提取音频为 MP3（有损但体积小）
ffmpeg -i "录制文件.mkv" -vn -acodec libmp3lame -b:a 320k "歌曲名.mp3"

# 提取音频为 FLAC（无损压缩，推荐）
ffmpeg -i "录制文件.mkv" -vn -acodec flac "歌曲名.flac"
```

#### 1.7 OBS 进阶技巧

| 技巧 | 操作 |
|------|------|
| **快捷键录制** | `设置 → 快捷键` → 搜索「开始录制」→ 设置如 `F6`，停止设为 `F7` |
| **降噪** | 右键音频源 → `滤镜` → 添加 `降噪` 滤镜（RNNoise 模式效果最好） |
| **增益/音量** | 右键音频源 → `滤镜` → 添加 `增益`，可放大或减小音量 |
| **只录特定应用** | Windows：添加「应用程序音频捕获」而非系统全局；Mac：暂不支持单应用，需静音其他应用 |
| **定时录制** | 使用 OBS 的 `脚本功能`（`设置 → 脚本`）添加 Lua 脚本实现定时启停 |
| **批量录制** | 录制多首歌曲后，用 ffmpeg 批量提取：`for f in *.mkv; do ffmpeg -i "$f" -vn -acodec flac "${f%.mkv}.flac"; done` |
| **MKV 转 MP4** | OBS 内置转换：`文件 → 重新封装录像` → 选择 mkv 文件 → 自动转为 mp4 |

### 方法 2：Audacity 录制（轻量方案）

适合只需要纯音频录制的场景，不需要视频功能。

1. **安装 Audacity**：
   ```bash
   brew install --cask audacity  # Mac
   ```

2. **Mac（需要 BlackHole）**：
   ```bash
   brew install blackhole-2ch
   ```
   - 打开「音频 MIDI 设置」→ 创建「多输出设备」→ 勾选扬声器和 BlackHole
   - 打开 Audacity → 录音设备选 BlackHole
   - 播放网易云歌曲 → 点击 Audacity 录制
   - 录完后 `文件 → 导出 → 导出为 MP3/WAV`

3. **Windows**：直接在 Audacity 中选择「Windows WASAPI」→ 选扬声器作为录音源即可。

### 方法 3：浏览器 + 插件（流式下载）

- 使用 Chrome 插件如 **Video DownloadHelper** 或 **Audio Downloader**
- 打开网易云网页版播放歌曲，插件会捕获音频流
- 点击下载保存

### 方法 4：直接下载工具

- 一些开源项目可以解析网易云链接直接下载，但要注意**版权问题**

> **注意**：录制/下载受版权保护的音乐仅供个人学习使用，不要用于商业用途或传播。

## 二、获取乐谱（Guitar / Bass / Drum）

### 在线乐谱网站

| 网站 | 特点 |
|------|------|
| **MuseScore** (musescore.com) | 免费/付费，有 Guitar、Bass、Drum 谱，支持下载 PDF 和 MuseScore 文件 |
| **Ultimate Guitar** (ultimate-guitar.com) | 最大的吉他谱库，也有 Bass 和 Drum tabs，免费版功能够用 |
| **Songsterr** (songsterr.com) | 在线播放 Guitar/Bass/Drum 谱，支持分轨显示 |
| **Guitar Pro Tabs** (guitarprotabs.org) | 提供 `.gp` 格式文件，可用 Guitar Pro 软件打开 |
| **E-Chords** (e-chords.com) | 和弦谱 +歌词，适合弹唱 |

### 推荐工具

1. **Guitar Pro**（付费）— 打开 `.gp` / `.gp3` / `.gp4` / `.gp5` 文件，支持 Guitar/Bass/Drum 多轨显示和播放
2. **MuseScore**（免费）— 开源乐谱编辑器，可打开 MuseScore 格式乐谱
3. **TuxGuitar**（免费）— Guitar Pro 的开源替代品，支持 `.gp` 格式

### 获取流程

```
1. 确定歌曲名称 + 原唱
2. 到上述网站搜索（如 "歌曲名 guitar pro" 或 "歌曲名 bass tab"）
3. 下载对应格式的谱子（PDF / GP / MuseScore）
4. 用 Guitar Pro 或 TuxGuitar 打开查看和练习
```

### 如果找不到现成谱子

- **MuseScore** 可以自己打谱（免费）
- **Guitar Pro** 可以自己编谱（付费但功能强大）
- **AnthemScore** 等 AI 工具可以从音频自动生成乐谱（准确度有限，需手动修正）

## 三、与 AI Music 生成结合的工作流

将网易云歌曲与 AI 音乐生成工具（如 Suno、Udio、Stable Audio 等）结合，可以实现参考学习 → AI 创作 → 再制作的完整闭环。

### 工作流总览

```
网易云歌曲 → 录制/提取素材 → 分析参考 → AI 生成 → 后期处理
```

### 步骤 1：录制/提取参考素材

1. 用前面介绍的方法录制你喜欢的网易云歌曲
2. 导出为 **WAV** 格式（无损，AI 工具识别更准确）
3. 如果只需片段，用 Audacity 裁剪出你想要的段落（如副歌 15-30 秒）

### 步骤 2：分析歌曲要素（用于 AI 提示词）

在生成前，先提取参考歌曲的关键信息：

| 要素 | 提取方法 | 示例 |
|------|---------|------|
| **BPM（速度）** | Audacity → `Analyze → Beat Finder`，或用 [tunebat.com](https://tunebat.com) 搜索 | 120 BPM |
| **调性** | [tunebat.com](https://tunebat.com) 或 Mixed In Key | C Major |
| **风格/流派** | 人工判断 | Synth Pop, Dream Pop |
| **乐器编制** | 参考乐谱或人工判断 | Electric Guitar, Bass, Drum Kit, Synth |
| **和弦进行** | 从吉他谱/和弦谱中获取 | I-V-vi-IV (C-G-Am-F) |
| **歌曲结构** | 人工标注 | Intro → Verse → Chorus → Bridge → Outro |

### 步骤 3：用 AI 工具生成音乐

根据不同 AI 工具的特性选择合适的方式：

#### Suno AI (suno.com)
```
提示词示例：
"Create a dream pop song in C major at 120 BPM.
Instruments: electric guitar arpeggios, deep bass, soft drum patterns, atmospheric synth pads.
Song structure: short intro, verse, big chorus with layered vocals, bridge, fade-out outro.
Mood: nostalgic, ethereal, bittersweet.
Style inspired by: [参考歌曲风格描述]"
```
- 支持上传音频作为参考（`Extend` 功能可以基于已有片段续写）
- 可指定歌词、风格、结构

#### Udio (udio.com)
```
提示词示例：
"dream pop, shoegaze influences, 120 bpm, female vocals,
jangly guitar, warm bass, reverb-heavy drums,
melancholic but uplifting, atmospheric production"
```
- 支持上传参考音频片段
- 可分段生成，精确控制歌曲结构

#### Stable Audio (stableaudio.com)
```
提示词示例：
"dream pop instrumental, 120 bpm, key of C major,
electric guitar arpeggios, bass guitar, drum kit with reverb,
ambient synth layers, ethereal atmosphere, 2 minute duration"
```
- 适合生成纯器乐/背景音乐
- 可精确控制时长

### 步骤 4：从 AI 生成结果提取乐谱

1. 导出 AI 生成的音频为 WAV/MP3
2. 用 **AnthemScore** 或 **Klangio** 等 AI 转谱工具将音频转为乐谱
3. 导入 **MuseScore** 或 **Guitar Pro** 进行编辑和修正
4. 分离出 Guitar / Bass / Drum 各轨

### 步骤 5：后期处理与再创作

1. **混合原始参考 + AI 生成**：在 Audacity 或 DAW（如 GarageBand、Logic Pro）中叠加
2. **手动修正 AI 乐谱**：AI 转谱通常需要人工调整细节
3. **重新录制**：用修正后的乐谱，自己演奏录制各乐器轨道
4. **混音**：将所有轨道在 DAW 中混合成最终作品

### 完整工作流示例

```
1. 在网易云找到喜欢的歌曲
2. 用 Audacity + BlackHole 录制 → 导出 WAV
3. 分析 BPM、调性、风格 → 写成 AI 提示词
4. 在 Suno/Udio 中输入提示词 → 生成 AI 版本
5. 对比原曲和 AI 版本，调整提示词重新生成
6. 满意后导出 AI 音频
7. 用 AnthemScore 转为乐谱 → 在 MuseScore 中修正
8. 导出 Guitar/Bass/Drum 分谱
9. 自己演奏录制或用 DAW 进一步编曲混音
```

### 实用技巧

- **迭代提示词**：第一次生成很少完美，多调整风格描述、乐器配比、BPM 等参数
- **分段生成**：先生成副歌部分确认风格，再生成完整歌曲
- **混合多个 AI 输出**：从 Suno 取旋律 + 从 Udio 取编曲，在 DAW 中组合
- **保留 MIDI**：如果 AI 工具支持导出 MIDI，比音频转谱准确得多
- **版权注意**：AI 生成的音乐如果明显模仿某首特定歌曲，商用可能存在版权风险

---

**总结**：录制推荐 **OBS Studio**（功能全面，Mac/Windows 通用）或 **Audacity + BlackHole**（轻量纯音频），乐谱推荐 **Ultimate Guitar + Guitar Pro** 的组合，AI 生成推荐 **Suno/Udio + AnthemScore** 的工作流，实现从参考歌曲到 AI 创作的完整闭环。
